# WinneAI — 제품 소개 발표 대본

> 대상: 비개발 임원진 / 톤: 앱 데모 소개 / 예상 시간: 12~15분

---

## 1. 오프닝

안녕하세요, 오늘 소개드릴 **WinneAI**는 S1000D 기술 교범 전용 AI 질의응답 시스템입니다.

한 마디로 정리하면, **기술 교범 전용 ChatGPT인데, 우리 서버 안에서만 돌아가는 시스템**입니다.

정비사가 "브레이크 교체 어떻게 해?"라고 한국어로 물으면, 영어 교범에서 관련 내용을 자동으로 찾아서 한국어로 정리해 줍니다. 외부 인터넷 연결 없이, PC 한 대에서 전부 동작합니다.

그럼 화면을 보면서 하나씩 소개드리겠습니다.

---

## 2. 접속 화면

웹 브라우저를 열고 서버 주소를 입력하면 바로 이 화면이 나옵니다.

별도 프로그램 설치가 필요 없습니다. 같은 네트워크에 있는 PC, 노트북, 태블릿 어디서든 접속할 수 있습니다.

화면 왼쪽이 **네비게이션 메뉴**, 가운데가 **대화 영역**입니다.

상단에 초록색 점과 함께 **"Model: EXAONE-3.5-7.8B · Ready"**라고 표시되면 AI 모델이 로딩된 상태, 바로 질문할 수 있다는 뜻입니다.

---

## 3. 질문하기

가운데 입력창에 질문을 한국어로 입력합니다.

예시 질문 카드가 3개 미리 준비되어 있어서, 처음 사용하는 분도 클릭 한 번으로 바로 체험할 수 있습니다.

- "자전거의 주요 구성품은 무엇인가요?"
- "브레이크 시스템 원리에 대해 알려줘"
- "앞바퀴 설치 절차를 알려줘"

질문을 입력하고 Send를 누르면, AI가 답변을 생성합니다.

---

## 4. 답변 확인

답변이 오면 이런 형태로 표시됩니다.

**AI 아이콘** 옆에 답변 본문이 나오고, 절차서 내용이면 **1단계, 2단계...** 형태로 정리되어 나옵니다. 정비사가 바로 따라 할 수 있는 형태입니다.

답변 아래에는 두 가지 정보가 추가로 표시됩니다:

1. **Copy 버튼** — 답변 텍스트를 클립보드에 복사합니다. 보고서나 정비 기록에 붙여넣기 할 때 유용합니다.

2. **추론 시간** — "추론 5.2s"처럼 AI가 답변을 만드는 데 걸린 시간이 표시됩니다.

---

## 5. 참고 문서 확인 (근거 추적)

답변 아래 **"참고 문서 (3건)"** 버튼이 있습니다. 클릭하면 스르륵 열리면서 근거 문서 목록이 나옵니다.

각 문서마다 세 가지 정보가 표시됩니다:

- **순위 뱃지** — #1, #2, #3 (가장 관련 높은 순서)
- **DMC 코드** — 원본 교범에서 해당 DM을 찾을 수 있는 고유 식별 코드
- **관련도 점수** — 퍼센트로 표시 (초록색 60% 이상, 노란색 45% 이상, 빨간색 미만)

이것이 왜 중요하냐면, **AI가 답변의 출처를 투명하게 보여준다**는 뜻입니다. "이 답변이 어느 문서에서 나온 건지" 바로 검증할 수 있습니다. AI를 맹신하는 것이 아니라, 근거를 확인하고 판단할 수 있는 구조입니다.

---

## 6. 후속 질문 (대화 맥락 유지)

한 번 질문하고 끝이 아닙니다. 이어서 후속 질문을 할 수 있습니다.

예를 들어:
> "브레이크 패드 교체 절차 알려줘" → (답변)
> "그리고 점검은 어떻게 해?" → (브레이크 점검 절차 답변)

"그리고"라고만 해도 이전에 브레이크에 대해 얘기했다는 맥락을 기억하고, 브레이크 점검 관련 문서를 자동으로 찾습니다.

최근 2회 대화까지 맥락을 유지합니다.

---

## 7. 세션 관리

왼쪽 화면 옆에 **Sessions 패널**이 있습니다. 토글 버튼(▶)을 누르면 열립니다.

여기서 **이전에 했던 모든 대화 목록**을 볼 수 있습니다. 날짜별로 그룹핑되어 있고, 각 세션에는 질문 제목과 시간이 표시됩니다.

- **세션 클릭** → 해당 대화 내용 전체를 다시 불러옵니다
- **X 버튼** → 개별 세션 삭제
- **휴지통 아이콘** → 전체 세션 일괄 삭제
- **검색창** → 세션 제목으로 검색

삭제 시에는 항상 **"정말 삭제하시겠습니까?"** 확인 팝업이 뜨기 때문에 실수로 지울 염려가 없습니다.

Chat 화면이 아닌 다른 메뉴로 이동하면 세션 패널은 자동으로 숨겨지고, Chat으로 돌아오면 다시 나타납니다.

---

## 8. New Chat

왼쪽 메뉴 상단의 **New Chat** 버튼을 누르면 새로운 대화를 시작합니다. 이전 대화와 완전히 분리된 새 세션이 만들어집니다.

주제가 바뀔 때, 예를 들어 브레이크에서 타이어로 질문 주제를 바꿀 때 사용하면 됩니다.

---

## 9. Settings 화면

왼쪽 메뉴에서 **Settings**를 클릭하면 설정 화면으로 이동합니다.

크게 세 영역이 있습니다:

### Search (Retrieval) — 검색 파라미터

이 영역의 슬라이더는 **사용자가 직접 조절**할 수 있습니다.

- **Top-K** — AI가 검색하는 후보 문서 수 (기본 10). 늘리면 더 넓게 찾지만 느려질 수 있습니다.
- **Relevance Threshold** — 최소 관련도 기준 (기본 0.30). 높이면 더 엄격하게 걸러냅니다.
- **Rerank Top-K** — 2차 검증 후 최종 선택 문서 수 (기본 3)
- **Max Context** — AI에게 전달하는 참고 문서의 최대 길이 (기본 10,000자)

일반 사용 시에는 기본값 그대로 두면 됩니다. 특정 상황에서 검색 범위를 넓히거나 좁히고 싶을 때 조절할 수 있습니다.

### LLM Generation — AI 모델 설정

이 영역은 **서버에서 설정된 값을 보여주는 읽기 전용** 카드입니다.

- **Temperature 0.20** — AI의 창의성 수준. 기술 문서이므로 낮게 설정하여 사실적 답변을 유도합니다.
- **Top-P 0.90** — 답변 다양성 제어
- **Repeat Penalty 1.05** — 같은 내용 반복 방지
- **Max Tokens 1,024** — 답변 최대 길이
- **Context Window 8,192** — AI가 한 번에 처리할 수 있는 텍스트 크기

### System Info

현재 시스템 상태를 보여줍니다:
- 사용 중인 **AI 모델명**
- **임베딩 모델명** (문서 검색 엔진)
- **인덱싱된 문서 수** (현재 156 chunks)

## 10. 한국어 질문 → 영어 교범 검색

이 기능은 화면에 직접 드러나지는 않지만, 내부에서 자동으로 동작하는 핵심 기능입니다.

기술 교범은 영어로 되어 있지만, 사용자는 한국어로 질문합니다.

시스템이 내부적으로 **한국어 키워드를 S1000D 영어 기술 용어로 자동 변환**합니다:

| 한국어 입력 | 자동 확장 |
|------------|----------|
| "브레이크 교체" | + brake remove install replace |
| "타이어 점검" | + tire tyre wheel check inspection |
| "기어 조정" | + gear transmission derailleur adjust |

약 60개의 정비 도메인 전용 용어쌍이 등록되어 있어서, 현장에서 쓰는 한국어 표현을 정확히 인식합니다.

---

## 11. 안전 정보 우선 검색

기술 교범에서 가장 중요한 것은 **안전 정보**입니다. WARNING, CAUTION 같은 경고문이요.

일반 AI 시스템에서는 이런 안전 경고가 긴 절차서 텍스트에 묻혀서 검색이 안 되는 경우가 많습니다.

WinneAI는 문서 등록 단계에서 **안전 경고문을 별도로 분리**해 둡니다. "브레이크 정비 시 주의사항"이라고 물으면, 절차서 전체가 아니라 해당 **경고문만 정확히** 찾아서 답변합니다.

---

## 12. 문서에 없는 질문 — 환각 방지

AI 시스템에서 가장 위험한 것은, 문서에 없는 내용을 그럴듯하게 **지어내는 것**입니다.

WinneAI는 이런 상황을 3단계로 나눠서 처리합니다:

- **관련 정보가 있을 때** → 문서를 인용하여 답변
- **직접 답은 없지만 관련 데이터가 있을 때** → 데이터를 인용하고 추론 (예: 정상 수치를 알려주며 비정상 상태를 설명)
- **전혀 관련 없을 때** → "제공된 문서에서 해당 정보를 찾을 수 없습니다." 한 줄만 답변

세 번째 경우가 핵심입니다. **모르면 모른다고 명확히 말하고, 절대로 일반 지식을 추가하지 않습니다.** 정비 현장에서 AI가 잘못된 정보를 줘서 사고가 발생하는 것을 원천 차단합니다.

---

## 13. 보안 — 완전 온프레미스

다시 한번 강조드리면, 이 시스템은 **인터넷 연결이 전혀 필요 없습니다.**

- AI 모델: 사내 PC에 저장
- 문서 데이터: 사내 PC에 저장
- 검색 엔진: 사내 PC에서 동작
- 웹 서버: 사내 PC에서 운영

외부 API 호출이 없으므로 **API 사용료도 없고, 월 구독료도 없습니다.** 초기 PC 구매 비용 외에 추가 운영 비용이 발생하지 않습니다.

보안 등급이 부여된 기술 교범도 안심하고 적용할 수 있습니다.

---

## 14. 확장 계획

현재는 S1000D Bike 샘플 데이터로 시연하고 있습니다.

- **실 장비 교범 적용**: XML 파일만 교체하면 바로 적용. 코드 수정 불필요
- **태블릿 앱 (개발 중)**: 정비 현장에서 태블릿으로 오프라인 사용
- **보안 등급별 접근 제어**: 사용자 권한에 따라 열람 가능 문서 자동 제한
- **답변 자동 검증**: AI 답변이 문서 내용과 일치하는지 2차 AI가 교차 검증

## 15. 마무리

WinneAI의 핵심 가치 세 가지입니다:

**첫째, 보안.** 데이터가 외부로 나가지 않습니다.

**둘째, 정확도.** S1000D 구조를 이해하는 검색 + 2중 검증 + 환각 방지.

**셋째, 편의성.** 한국어 질문 한 줄로, 영어 교범에서 즉시 답변.

감사합니다.
